## Getting Started

### Using as a dependency

```sh
npm install cubo-ui
```

#### app.module.ts

```ts
...
import { CbAvatarModule } from 'cubo-ui';
...

@NgModule({
  declarations: [
    AppComponent,
    ...
  ],
  ...
  imports: [
    ...
    CbAvatarModule
  ],
  ... 
})
```
